# Multi-Role Admin Dashboard

A complete role-based admin dashboard built with React, featuring authentication, CRUD operations, charts, and dark mode.

## 🚀 Features

- **Authentication** - Mock login system with 3 roles
- **Role-Based Access** - Admin, Manager, User roles with different permissions
- **Dashboard** - Statistics cards and interactive charts
- **CRUD Operations** - Manage users and products
- **Search & Filter** - Search users and filter products
- **Pagination** - Navigate through large datasets
- **Dark/Light Mode** - Theme toggle with persistence
- **Fully Responsive** - Works on all devices
